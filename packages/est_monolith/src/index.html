By default without any plugin specified, Google search will be used, unless the query contains CJK characters, then Bing CN will be used.

Use @w.zh and @w.ja for Chinese and Japanese Wikipedia respectively.

Use ! to perform a DuckDuckGo Bang Search, and \ to jump to the first DuckDuckGo search result.
